# Liuvis Studio

AI 驱动的 3D 智能设计平台

## 简介

Liuvis Studio 是一款基于大语言模型（LLM）的 3D 智能设计平台，用户通过自然语言对话即可完成从概念到三维模型的全流程创建，大幅降低了 3D 建模的技术门槛。

## 核心亮点

### 1. 自然语言驱动设计

用户只需输入如"创建一个蓝色金属立方体"等自然语言描述，系统通过 NLU（自然语言理解）引擎解析用户意图，自动识别创建、修改、查询等操作类型，并驱动后续的设计与生成流程，无需掌握任何专业建模工具。

### 2. LLM + 程序化几何生成

平台采用"LLM 场景规划 + 程序化几何构建"的混合架构。大模型负责理解语义、规划场景布局和材质属性，程序化几何引擎则将抽象描述精确转化为真实的三维网格数据（顶点、法线、UV、索引），支持 Box、Sphere、Cylinder、Cone 等基础图元的参数化生成。

### 3. 多格式输出

支持 GLB（实时预览）、STL（3D 打印）、OBJ（通用交换）、STEP（CAD 工程）四种主流 3D 格式的生成与下载。用户在模型预览界面可随时切换目标格式导出，满足从 Web 展示到工业制造的不同场景需求。STEP 格式通过手写 AP214 B-Rep 拓扑结构实现，可被 SolidWorks、FreeCAD、Fusion 360 等主流 CAD 软件导入。

### 4. 实时 3D 预览

基于 Three.js 的 WebGL 3D 查看器，支持鼠标旋转、缩放、平移等交互操作，模型生成后即可在浏览器中实时渲染预览，所见即所得。

### 5. 对话式迭代修改

支持对已生成模型进行对话式修改，如"把颜色改成红色""放大两倍"等，系统自动识别修改类型（颜色、材质、尺寸、变换），应用变更后重新生成模型，实现快速迭代。

### 6. 知识库与模型管理

内置向量知识库，自动索引已生成的模型，支持语义检索复用历史设计。模型管理页面提供搜索筛选、格式切换下载、一键删除等功能，方便用户管理设计资产。

### 7. 实时流式交互

通过 SignalR 和 SSE 实现思考过程、生成进度的实时流式推送，用户可随时观察 AI 的推理过程，提升交互透明度和信任感。

## 技术架构

- **前端**：Blazor Server + MudBlazor 组件库
- **后端**：.NET 10 分层架构（Core / Generation / NLU / Design / Infrastructure）
- **3D 渲染**：Three.js (WebGL)
- **AI 引大**：LLM 驱动的场景规划与语义理解
- **几何引擎**：自研程序化几何构建器，支持 GLB/STL/OBJ/STEP 多格式输出
- **实时通信**：SignalR + SSE 流式推送
- **知识库**：向量检索，支持模型语义搜索与复用
